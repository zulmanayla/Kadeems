# KDM RANK
KDM Rank adalah aplikasi dashboard monitoring dan evaluasi capaian KDM yang digunakan untuk memantau perolehan tagging usaha per desa dan kecamatan secara real-time, terstruktur, dan interaktif, berbasis data Google Spreadsheet. Aplikasi ini dirancang untuk membantu pengelola lapangan, koordinator wilayah, dan pengambil keputusan dalam memantau progres pelaksanaan KDM, mengidentifikasi desa yang sudah atau belum tuntas, serta mencatat fenomena lapangan sebagai bahan evaluasi untuk persiapan SE2026.
